# SkillMach AI - Smart Training Machine Management

SkillMach AI is a data-driven framework designed for the efficient management of training machinery across multiple skill development centers. It leverages Generative AI to provide predictive maintenance, strategic resource allocation, and real-time operational support.

## 🚀 Key Features

- **AI Zaya Assistant:** A sophisticated AI chatbot for troubleshooting, machine discovery, and voice-enabled support.
- **Predictive Maintenance:** AI-powered health monitoring that analyzes sensor data to avert equipment failure.
- **Dynamic Analytics:** Real-time visualizations of machine utilization, center demand, and inventory health.
- **Smart Booking:** Reservation system for managing student access to high-value training equipment.
- **Fleet Management:** Centralized registry for tracking machinery across various geographic locations.
- **Mobile-First Design:** Fully responsive interface with a side-drawer navigation for on-the-go management.

## 🛠️ Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **AI Engine:** Google Genkit (Gemini 2.5 Flash)
- **UI Components:** Shadcn UI & Radix UI
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Charts:** Recharts
- **Animations:** Tailwind Animate

## 🤖 AI Flows (Genkit)

- `aiZayaOperationalSupport`: Handles real-time student queries and troubleshooting.
- `predictiveMaintenanceInsightGen`: Generates technical explanations for predicted machine failures.
- `strategicMachineAllocationRecs`: Optimizes machine placement across centers based on demand.
- `aiZayaInfoLearningAssistant`: Summarizes manuals and provides general training information.

## 📂 Project Structure

- `src/app/dashboard`: Core application features and views.
- `src/ai/flows`: Logic for Genkit-powered AI operations.
- `src/components/ui`: Reusable high-fidelity UI components.
- `src/lib/mock-data.ts`: Centralized simulation data for the fleet and centers.
